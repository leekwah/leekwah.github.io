# SDKMAN

## MacOS에서는 SDKMAN을 통해서, Java의 버전을 관리할 수 있다.

```terminal
$ sdk version
$ sdk version
$ sdk list java
$ sdk install java 11.0.16-amzn
$ sdk default java 11.0.16-amzn
$ sdk use java 11.0.16-amzn
$ sdk current java
$ echo $JAVA_HOME
```
