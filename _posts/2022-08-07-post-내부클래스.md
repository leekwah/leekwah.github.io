# 내부클래스(inner class)

클래스 안의 클래스

```java
class A {
  ...
}
class B {
  ...
}
```

```java
class A { // B의 외부 클래스
  ...
    class B { // A의 내부 클래스
      ...
    }
}
```

### 내부 클래스의 장점

내부 클래스에서 외부 클래스의 멤버들을 쉽게 접근할 수 있다.
코드 복잡성을 줄일 수 있다. (캡슐화)
